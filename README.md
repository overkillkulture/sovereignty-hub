### Offline AI Stack

Run your own AI assistant on a Raspberry Pi 5 (~$75):

| Component | Tool | What it does |
|-----------|------|-------------|
| Speech-to-Text | Whisper (in-browser or local) | Converts your voice to text, no internet |
| Language Model | Ollama (Phi-3 / Mistral 7B) | Thinks and responds locally |
| Text-to-Speech | Kokoro TTS | Near-human voice, runs on CPU, Apache 2.0 |
| Mesh Radio | Meshtastic (Heltec V3) | Communication without cell towers |
| Brain | Cyclotron (SQLite) | 174K+ knowledge atoms, searchable |

Total cost: ~$150 for a complete offline AI node with mesh radio.

### How to Contribute

1. **Pick a pillar** that matches your skills
2. **Read the blueprint standard** (`BLUEPRINT_STANDARD.md`)
3. **Fork this repo** and submit a PR with your blueprint
4. **Join the conversation** — [conciousnessrevolution.io](https://conciousnessrevolution.io)

Every blueprint follows the same format:
- What it solves
- Parts list with prices and links
- Step-by-step build instructions
- Safety warnings
- How to test it works
