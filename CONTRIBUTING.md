# How to Contribute

## The Rule
Every hub that uses this blueprint can submit improvements back so every other hub benefits. Fix what's broken. Share what works. Document what doesn't.

## What to Contribute

### Easy (5 minutes)
- Fix a broken supplier link or update a price
- Correct a factual error
- Add a product you tested and recommend

### Medium (30 minutes)
- Add your region's specifics (climate, local suppliers, regulations)
- Document a product you built with photos and build cost
- Share real revenue data (anonymized if needed)

### Deep (Hours)
- Write a new pillar variation for your region
- Design and share an STL file for a 3D-printed product
- Document a failure — what you tried that didn't work and why
- Create a new archetype path for a community member type we missed

## How to Submit

1. Fork this repository
2. Make your changes in your fork
3. Submit a Pull Request with a clear description of what changed and why
4. Tag your PR with the relevant pillar number (e.g., `pillar-01-water`)

## What We Don't Accept
- Marketing or promotional content for specific brands
- Changes that remove the CC BY-SA license
- Content that advocates illegal activity
- Unverified health claims (medical pillar requires sources)

## The Knowledge Loop
Build → Document → Share → Improve → Repeat. The loop never closes. The blueprint never finishes. It just gets better.
